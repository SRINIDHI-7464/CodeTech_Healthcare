1. Introduction

Healthcare Patient Metrics is an important application of Data Analytics that helps healthcare organizations analyze patient information, monitor health trends, and improve medical decision-making. By studying patient demographics, medical conditions, hospital admissions, billing information, and treatment records, healthcare providers can enhance the quality of patient care and optimize hospital management.

In this project, Python is used to analyze healthcare patient data obtained from the Healthcare Dataset. The dataset contains patient details such as age, gender, blood type, medical condition, admission type, hospital, doctor, billing amount, and other healthcare-related information. Data preprocessing, exploratory data analysis (EDA), and visualization techniques were applied to understand patient characteristics and healthcare trends. This project demonstrates how healthcare analytics can support better patient management and improve healthcare services.

2. Objective

The main objectives of this project are:

* Analyze healthcare patient information
* Perform data cleaning and preprocessing
* Study patient demographics and medical conditions
* Visualize healthcare trends and hospital statistics
* Analyze billing and admission patterns
* Generate meaningful insights for healthcare management

3. Dataset Description

The dataset used for this project was downloaded from Kaggle and contains healthcare patient records collected from hospitals.

The dataset includes the following attributes:

* Name
* Age
* Gender
* Blood Type
* Medical Condition
* Doctor
* Hospital
* Admission Type
* Billing Amount
* Insurance Provider
* Medication
* Test Results

The dataset is stored in CSV format and is widely used for healthcare analytics, patient monitoring, and medical data analysis.

Dataset Name: Healthcare Dataset

Dataset Source:

https://www.kaggle.com/datasets/prasad22/healthcare-dataset
